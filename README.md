# 💰 Finance Tracker Web App

A secure and user-friendly finance tracker that helps users manage **ledgers, transactions, and budgets** in one place.  
Each user’s financial data is isolated and protected using Firebase authentication.

🔗 **Live App:**  
https://finance-tracker-vfjotjafcgnlht4sftjthq.streamlit.app/

---

## 🚀 Features

### 📒 Ledgers
- Create and manage multiple ledgers (e.g. Cash, Bank)
- Organized financial records
- Running balance for better visibility

### 💸 Transactions
- Add income and expense transactions
- Date-wise transaction history
- Automatic balance updates
- Ledger-based transaction tracking

### 📊 Budget Management
- Set budgets for different categories
- Track spending against allocated limits
- Helps users control overspending

---

## 🔐 Data Privacy & Security

- Each user has a **unique account (UID)**
- User data is stored separately in Firebase
- No user can view or modify another user’s data
- Secure database rules enforced

---

## 🛠 Tech Stack (High-Level)

- **Frontend:** Streamlit  
- **Backend:** Python  
- **Database:** Firebase  
- **Authentication:** Firebase Auth  
- **Deployment:** Streamlit Cloud  
- **AI Assistance:** Gemini CLI  

---

## 👤 User Flow

1. User creates an account / logs in  
2. User creates ledgers  
3. Transactions are added to selected ledgers  
4. Budgets are set and monitored  
5. All data is saved securely per user
