# Budget Management Feature

This feature will allow users to set monthly budgets for categories and track spending against them.

## Features to Build

- **Set Budget**: Allow users to set a monthly budget amount for a specific category (Food, Transport, Shopping, Bills, Entertainment, Health).
- **View Budgets**: Display a Rich table showing:
    - Category
    - Budget Amount
    - Spent (current month)
    - Remaining
    - Utilization % (with progress bar)
    - Status (OK / Warning / Over)
- **Color Coding**:
    - Green: Under 70% used
    - Yellow: 70-100% used
    - Red: Over 100% (overspending)
- **Overall Budget Summary**: Display total monthly budget, total spent, total remaining, and overall utilization %.
- **Recommendations**: Provide recommendations for categories over budget.
